Project Description:
This program brings value to the user by providing detailed information about specific animals, like dog, fox, elephant etc.

Usage:
Before usage, each user needs to register at:
https://api-ninjas.com/register
Here the user can download or copy the API KEY.
This key needs to be stored in the .env file in the folder where the
app is also stored. 
The entry in the .env file should like as follows:
API_KEY='XXXXXXxxxxxxYYYYYYYyyyyyyy'
Without any spaces in front and after the '='-sign

On request, type an animal name. The detailed information will be stored in a file called 'animal.html'
